Program

    Tuesday 22nd June – Popular science talks in Polish

    • 17:00 - 17.45 Popular science talk (in Polish) „Teoria węzłów w historycznej perspektywie" ("Knot Theory in Historical Perspective") by Józef Przytycki, The George Washington University
    • 17:45 - 18:30 "Knots on the border of physics, chemistry and biology" – discussion (in Polish) by Przytycki, Minor, Cieplak, Niemyska, Sulkowska

    • 15:00 - 15:45 Popular science talk: "The world of proteins -- what do we know about them" by Sophie Jackson, Cambridge University

    • 16:15 - 17:00 coffee break

    Knotted proteins – KnotProt + KnotPull

    • 14:00 - 14:30 General introduction to knotted proteins (by Ken Millett)
      • hands on workshop:
    • 14:30 - 15:00 KnotProt Database – how to search data, find similar topologies, etc. (by Joanna Sulkowska/Szymon Niewieczerzal)

    • 15:15 - 15:45 KnotProt Server – how to run your own analysis (by Paweł Rubach/Szymon Niewieczerzał)

    • 16:15 - 16:30 coffee break

    • 16:30 - 17:00 General introduction to proteins with links (by Przytycki)
      • hands on workshop:
    • 17:00 - 17:30 LinkProt Database and Server – how to search data, find similar topologies, run your own analysis (by Joanna Sulkowska/Szymon Niewieczerzal)
    • 17:30 - 18:00 coffee break

    Lasso proteins – LassoProt + PyLasso

    • 18.40 - 19:00 coffee break
      hands on workshop:

    • 19:20 - 19:40 LassoProt Server – how to run your own analysis (by Wanda Niemyska)
    • 19:40 - 20:00 PyLasso – how to use the PyMol plugin (by Bartosz Greń)

    Topoly – Python package to study topology of proteins and other (bio)polymers

    • 19:00 - 19:30 Some examples how to use Topoly II (by Wanda Niemyska/Paweł Rubach)

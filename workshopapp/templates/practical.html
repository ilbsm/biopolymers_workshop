Topoly

Topoly is Python 3 package for topological analysis of (bio)polymers.
Before instalation make sure you have a recent version of pip by typing in terminal:
pip3 install --upgrade pip
Then install by typing:
pip3 install topoly
More information at Topoly package webpage.

Knot_Pull

Knot_Pull is Python 3 package for simplifying a 3D structure with perserving its topology.
Before instalation make sure you have a recent version of pip by typing in terminal:
pip3 install --upgrade pip
Then install by typing:
pip3 install topoly
More information at Knot_Pull github repository

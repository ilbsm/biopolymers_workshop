Topoly

Topoly is a Python 3 package for the topological analysis of (bio)polymers.
It works on Mac OS X and Linux.
Before installation make sure you have a recent version of pip by typing in terminal:
pip3 install --upgrade pip
Then install by typing:
pip3 install topoly
More information on the Topoly package webpage.

Knot_Pull

Knot_Pull is Python 3 package for simplifying a 3D structure while preserving its topology.
Before installation make sure you have a recent version of pip by typing in terminal:
pip3 install --upgrade pip
Then install by typing:
pip3 install knot_pull
More information on the Knot_Pull github repository

Genus for biomolecules

Genus for biomolecules is a database that collects information about topological structure and complexity of proteins and RNA chains, which is captured by the genus of a given chain and its subchains. For each biomolecule this information is shown in the form of a genus trace plot, as well as a genus matrix diagram. Such information is assembled for all protein and RNA structures deposited in the PDB.

Knot_Pull

Knot_Pull is Python 3 package for simplifying a 3D structure while preserving its topology.
Before installation make sure you have a recent version of pip by typing in terminal:
pip3 install --upgrade pip
Then install by typing:
pip3 install knot_pull
More information on the Knot_Pull github repository
